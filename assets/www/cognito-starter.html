Amazon Cognito Starter Code

If you haven't already, download and install the AWS Mobile SDK.

  • Android
  • iOS

Android Starter Code

Step 1: Import the Cognito libraries into your class:

import com.amazonaws.android.auth.CognitoCredentialsProvider;

Step 2: Initialize the Cognito client:

CognitoCredentialsProvider cognitoProvider = new CognitoCredentialsProvider(
    myActivity.getContext(), // get the context for the current activity
    "324549020557",
    "us-east-1:9be90663-dfb3-4b34-9871-4e83ef88d19a",
    "arn:aws:iam::324549020557:role/Cognito_28DaysUnauth_DefaultRole",
    "arn:aws:iam::324549020557:role/Cognito_28DaysAuth_DefaultRole"
);

Step 3: Retrieve your identity ID:

cognitoProvider.getIdentityId();

iOS Starter Code

Step 1: Import the Cognito header files:

If you a using the SDK in a framework format:

#import <AWSiOSSDKv2/AWSCore.h>

If you are using the SDK in CocoaPods:

#import "AWSCore.h"

Step 2: Initialize the Cognito client:

AWSCognitoCredentialsProvider *credentialsProvider = [AWSCognitoCredentialsProvider 
   credentialsWithRegionType:AWSRegionUSEast1
   accountId:@"324549020557"
   identityPoolId:@"us-east-1:9be90663-dfb3-4b34-9871-4e83ef88d19a"
   unauthRoleArn:@"arn:aws:iam::324549020557:role/Cognito_28DaysUnauth_DefaultRole"
   authRoleArn:@"arn:aws:iam::324549020557:role/Cognito_28DaysAuth_DefaultRole"];
      
AWSServiceConfiguration *configuration = [AWSServiceConfiguration configurationWithRegion:AWSRegionUSEast1
   credentialsProvider:credentialsProvider];
   
[AWSServiceManager defaultServiceManager].defaultServiceConfiguration = configuration;

Step 3: Get your identity ID:

[[credentialsProvider getIdentityId] continueWithSuccessBlock:^id(BFTask *task){
   NSString* cognitoId = credentialsProvider.identityId;
   return nil;
}];

// once you receive an identity id you can simply access it through the identityId property
NSString* cognitoId = credentialsProvider.identityId;
